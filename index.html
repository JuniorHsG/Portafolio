Predicción de Ventas - Dulcería y Botanas

Este sistema fue desarrollado como parte de un proyecto de titulación con el objetivo de predecir las ventas semanales de productos en una tienda de dulcería y botanas. La herramienta ayuda a visualizar la demanda estimada y facilitar decisiones sobre el inventario.

Arquitectura: Es una aplicación web sencilla que utiliza un modelo de regresión polinomial implementado en Python, con una base de datos local en SQLite y una interfaz construida con HTML, CSS y JavaScript.
